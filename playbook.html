05 スクリプト設計
キーワード設定と出口の優先順位
✅ 私が担当者
私です / 私が担当者 / 僕が担当者 / 担当者は私 / オーナーです / 私がオーナー / オーナーになります
🚫 興味なし
お断り / 興味な / 興味無 / 興味ありません / 結構です
⚠️ 解析不能（受け皿）
上記いずれにもヒットしない発言すべて → 解析不能プロセスへ移行
❌ 悪い設定
優先①: 私が担当者
優先②: 興味なし

「私が担当ですがお断りです」
→「私が担当者」ヒット → 転送❌
✅ 正しい設定
優先①: 興味なし
優先②: 私が担当者

「私が担当ですがお断りです」
→「興味なし」ヒット → 断り✅
⚠️ 左上にあるほど優先度が高い。複合発言（担当+拒否）に備え「興味なし」を必ず上位にする。

📋 通話ログを確認
発信タスク → 通話詳細 → 通話記録で文字起こしを確認。解析不能になった発言を特定する。
🔤 キーワードを追加
「私がオーナーです」など未登録表現を「私が担当者」グループに追加。正規表現も活用。
🗂️ 解析不能プロセス作成
どの出口にもヒットしない発言の受け皿プロセスを作成。再質問で担当者確認を促す。
↕️ 優先順位を調整
複合発言（担当+拒否）への対応で「興味なし」を「私が担当者」より上位に設定する。
✅ 編集後は「保存」と「公開」の両方を必ず押す。公開しないと変更が反映されない。
10 FAQ & ツールURL集
現場担当者向け FAQ
Q 顧客にAIだとバレませんか？
人間の音声を録音して使用。業界別スクリプトでカスタマイズしており、自然な受け答えが可能です。
Q 人間のオペレーターは不要になりますか？
AIはスクリーニングを担当、有望な見込み客のみ担当者に引き継ぎます。クロージングは人間が対応します。
Q 1日に何件発信できますか？
最大8回線同時架電。1名で月3〜10万コールが可能（従来比約10倍）。24時間365日稼働します。
Q 架電結果はどこで確認できますか？
管理画面（robot.topsaler.cc）でリアルタイムにデータ確認・分析が可能です。週次レポートも提供されます。
Q 問題が発生したときの連絡先は？
ChatWork「（株）アプクロ×AIテレサポご案内」にて担当者（李様・山田様・青木様）へ即座に連絡してください。
Q スクリプトはいつ変更できますか？
管理画面から随時変更可能。SoloptiLink担当者との月次MTGでもスクリプト改善の伴走サポートがあります。
